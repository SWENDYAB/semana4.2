Selector Universal para Box Model

El asterisco (*) es el selector universal en CSS, con el box model hacen que todos sus selectores mantengan su ancho o alto sin que sean afectados por el padding o border

Ejemplo: * { box-sizing: border-box; }

Margin Negativo

El margen se establece en un valor negativo, cuando la caja no sólo no tiene un espacio extra hacia fuera, sino que tampoco tiene su espacio básico y se coloca invadiendo el espacio delimitado por otra caja contigua. margin negativo sería “Cubrir total o parcialmente” con elementos adyacentes.

Ejemplo: margin-top: -10px
